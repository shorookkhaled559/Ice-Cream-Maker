#  Ice Cream Maker

This project is an **interactive web application** that allows users to
**create and customize ice cream scoops** with different colors and
toppings.
It aims to provide a **fun, simple, and creative experience** entirely
inside the browser.

##  Features

-   **Change Scoop Colors** --- Each scoop changes color every time you
    click its button.
-   **Add Toppings** --- Two topping styles can be applied dynamically
    to scoops.
-   **Interactive & Animated** --- Smooth color transitions and playful
    UI.
-   **Lightweight** --- Fully frontend, no backend required.

------------------------------------------------------------------------

##  Technologies Used

-   **HTML5** --- Page layout and structure
-   **CSS3** --- Styling for scoops, bowl, and effects
-   **JavaScript (ES6)** --- Handles all interactions and dynamic color
    changes
-   **Google Fonts** --- Poppins for a clean and modern UI

##  How It Works

1.  Each scoop (`scoop1`, `scoop2`, `scoop3`) starts with default
    styling.
2.  When the user clicks a scoop button, JavaScript:
    -   Applies a new background color from the predefined array.
    -   Moves to the next color using a rotating counter.
3.  When clicking topping buttons:
    -   JavaScript adds a **radial-gradient** to simulate toppings.
4.  All changes appear instantly --- no reload needed.

------------------------------------------------------------------------

##  Getting Started

1.  **Clone or download** this repository.
2.  Open `index.html` directly in your browser.
3.  Click the buttons to:
    -   Change scoop colors
    -   Add toppings
4.  Enjoy your custom ice cream!
